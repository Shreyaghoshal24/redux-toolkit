# Redux Toolkit POC

A simple React application that demonstrates how to use Redux Toolkit for state management. It allows you to add items to a list and displays the list of items in real-time.

## Tech Stack

- **Client:** React, Redux Toolkit
- **Styling:** Custom CSS
